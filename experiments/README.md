# Experiments

Exploratory research and proof-of-concept features for future EmoCore releases.

## Active Experiments

- `vector_influence/`: Testing if governance can be influenced by raw hidden-state vectors rather than scalar signals.
- `multi_agent_governance/`: Synchronizing halts across a swarm of agents sharing a global pressure state.

> [!WARNING]
> Code in this directory is experimental and may violate EmoCore invariants. Not for production use.
